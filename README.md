# 坦克大战 (Battle City)

一款可自行部署、在浏览器中游玩的经典坦克射击游戏。纯 HTML/CSS/JS 实现，无需任何框架或构建工具。后续将逐步修复部分bug,完善并丰富游戏内容。预计将上线选关功能，双人游戏功能。同时将尝试发布可安装的.apk版本

---

## 快速开始

直接用浏览器打开 `index.html` 即可开始游戏，或部署到任意静态服务器。

---

## 操作说明

| 按键 | 功能 |
|------|------|
| `↑` `↓` `←` `→` / `W` `A` `S` `D` | 移动坦克 |
| `空格` / `J` | 射击 |
| `P` | 暂停 / 继续 |
| `Enter` | 开始游戏 / 重新开始 |

---

## 游戏规则

保护底部的**基地（鹰标）**不被摧毁。消灭当前关卡所有敌方坦克即可进入下一关。基地被摧毁或生命耗尽则游戏结束。

---

## 核心功能

### 20 大关卡 · 渐进难度

游戏内置 20 张手工设计的地图，难度从低到高分为 5 个梯度：

| 梯度 | 关卡 | 难度特征 |
|------|------|----------|
| 极简单 | 1-3 | 训练场、砖墙堡垒、安全走廊 — 少量砖墙，开阔射击通道 |
| 简单 | 4-6 | 草丛掩护、河畔前线、双塔防线 — 引入草丛和河流地形 |
| 中等 | 7-10 | 玻璃初现、迷宫小径、水晶庭院、丛林深处 — 玻璃障碍出现 |
| 困难 | 11-13 | 十字路口、铁壁合围、河湾峡谷 — 多层防线，复杂通道 |
| 极难 | 14-20 | 开阔地带、玻璃要塞…最终决战 — 大量玻璃墙，基地几乎无保护 |

每关的敌人数量、最大同时在场数、生成间隔随关卡递增，最高 28 个敌人、7 个同时在场。

### 5 种地形

| 地形 | 特性 |
|------|------|
| **砖墙** | 可被子射击毁 |
| **钢墙** | 不可摧毁 |
| **玻璃** | 仅 3 级坦克子弹可击穿，普通子弹被阻挡 |
| **草丛** | 视觉遮挡，坦克和子弹可穿过（覆盖在所有实体上方渲染） |
| **河流** | 装饰性水域，无实际阻挡效果 |

### 坦克升级系统

玩家坦克有 **3 个等级**，通过拾取进化道具提升：

| 等级 | 射击冷却 | 特殊能力 |
|------|----------|----------|
| ★☆☆ | 15 帧 | 基础 |
| ★★☆ | 10 帧 | 坦克下方显示一颗星 |
| ★★★ | 6 帧 | 可击穿玻璃障碍，坦克下方显示两颗金色星 |

### 敌方 AI

- **普通坦克**（红色）：速度较慢，100 分
- **快速坦克**（绿色）：速度更快，200 分，出现概率随关卡提升
- 敌人随机移动，有 18% 概率朝玩家方向追踪，靠近基地时倾向横向移动
- 敌人之间、敌人与玩家之间均有碰撞检测

### 5 种道具（能量果实）

击杀敌人有 **22%** 概率掉落道具，出现在地图空地砖块上，持续约 10 秒后消失：

| 道具 | 图标 | 概率 | 效果 |
|------|------|------|------|
| **进化** | ★ | 32% | 提升坦克 1 级（满级时 +200 分） |
| **生命** | ♥ | 20% | +1 条命 |
| **炸弹** | ● | 16% | 炸毁场上所有敌方坦克 |
| **无敌** | ⛑ | 16% | 4 秒无敌时间，闪烁提示 |
| **护盾** | 🛡 | 16% | 基地砖墙 → 钢墙（不可摧毁），效果仅限当前关 |

### 生命与重生

- 初始 **3 条命**，通过生命道具可增加
- 玩家被击毁后在出生点重生，附带 **2.5 秒无敌时间**（闪烁效果）
- 生命耗尽则游戏结束

### 音效系统

- **背景音乐**：循环播放战场音乐 (`war1.wav`)
- **音效覆盖**：射击声、移动声（含 170ms 节流）、爆炸声、道具拾取音、游戏结束音
- **降级机制**：音频文件加载失败时自动使用 Web Audio API 合成音效 (beep fallback)
- 首次用户交互时初始化 AudioContext

### 视觉特效

- 子弹击中墙壁/坦克的**爆炸动画**（小爆炸 11 帧、大爆炸 16 帧）
- 道具**浮动动画**（上下 bob 效果）
- 无敌状态**闪烁效果**
- 菜单画面装饰性敌方坦克随正弦波浮动
- 暗色主题 + 霓虹绿 HUD，像素风格渲染 (`image-rendering: pixelated`)

### HUD 信息面板

- 当前分数
- 剩余敌人数
- 关卡编号 + 地图中文名称
- 坦克等级（★ 星级显示，颜色随等级变化）
- 生命数（心形图标展示）
- 底部操作键位提示

### 响应式布局

- 桌面端 640×640 游戏画布
- 移动端自适应缩放（`aspect-ratio: 1`），HUD 和控制提示字体缩小

## 技术架构

```
tank-game/
├── index.html          # 入口页面，按依赖顺序加载 JS
├── style.css           # 全局样式、HUD、响应式
├── js/
│   ├── constants.js    # 常量定义、素材路径、图片预加载
│   ├── maps.js         # 20 张关卡地图数据
│   ├── audio.js        # 音频管理、音效合成降级
│   ├── entities.js     # Tank、Bullet、Explosion、Fruit 类
│   ├── rendering.js    # Canvas 渲染（地图/实体/菜单/覆盖层）
│   └── game.js         # 游戏主循环、状态管理、输入处理
├── images/             # 精灵图、地形贴图、爆炸帧动画（GIF/PNG）
└── audio/              # 背景音乐、音效文件（WAV）
```

纯原生 JavaScript，无第三方依赖。6 个 JS 文件按依赖顺序通过 `<script>` 标签加载，所有类/函数挂载全局作用域。
